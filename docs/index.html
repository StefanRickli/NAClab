NAClab

Welcome to the NAClab project page! This repository is dedicated to preserving the Numerical Algebraic Computing Toolbox for MATLAB, originally developed by Dr. Zhonggang Zeng and collaborators. NAClab offers powerful tools for solving advanced numerical algebraic problems, ranging from polynomial computations to nonlinear system solving, with applications in scientific research and engineering.

About NAClab

NAClab is a comprehensive research toolbox that excels at:

  • Solving hypersensitive problems from empirical data.
  • Performing intuitive WYSIWYG polynomial computations in MATLAB.
  • Directly solving rank-deficient linear operator equations.
  • Handling nonlinear systems with singular Jacobians and non-isolated solutions.

This repository is a community-driven effort to preserve and share the NAClab software and its documentation, ensuring its continued accessibility after the original website became unreachable. Contributions of missing documents, images, or code fragments are highly welcome.
