NAClab

Welcome to the NAClab project page. This repository aims to preserve the Computing Toolbox for Matlab, originally created by Prof. Zhonggang Zeng. Unfortunately, the original website is no longer accessible, and this project serves as a community service to make the code and accompanying documentation available once again.
